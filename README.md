# dotHtml-2-dotJade
Html to Jade files convertor, based on html2jade 0.8.4

# usage
Create your project's directory

    mkdir your_dir

Move to your directory

    cd your_dir
    
Clone the project

    git clone https://github.com/alekaizer/dotHtml-2-dotJade.git

Install packages

    npm install

Run the app

    node server.js
